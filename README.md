# PvAI

A browser-based game concept built as part of a personal front-end portfolio, focused on player-versus-AI interaction, progression, and dynamic gameplay.

## Live Demo

https://laurandreea10.github.io/PvAI/

## Features

* Player versus AI gameplay concept
* Interactive browser experience
* Progress-oriented game structure
* Dynamic front-end presentation
* Responsive layout
* GitHub Pages deployment

## Technologies

* HTML
* CSS
* JavaScript
* GitHub Pages

## Purpose

This project was created as a personal front-end experiment exploring player interaction, AI-inspired mechanics, and game progression in a web-based format.
